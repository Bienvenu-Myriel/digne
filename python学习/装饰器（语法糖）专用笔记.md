基础：
def mygo(func):

    def wrapper():
        print("函数开始")

        func()

        print("函数结束")

    return wrapper
使用：

@mygo
def hello():
    print("hello world")

hello()

结果：
函数开始
hello world
函数结束



真实开发里一般写成这样：

def mygo(func):
    @functools.wraps(func)
    #保护被装饰函数的“身份信息”不被丢失的代码
    def wrapper(*args, **kwargs):
    #这是一个 内部函数。这里用到了两个特殊参数：*args **kwargs 全世界的程序员基本都用 **kwargs，它就像是一个无限大的带标签的快递箱
    #*args + **kwargs = 装饰器的万能接口
        print("函数开始")

        result = func(*args, **kwargs)
       #*args会把（3，5）拆开成3，5把参数传给原函数add（）

        print("函数结束")

        return result
        把结果返回以让外部调用

    return wrapper
    最关键的一句：mygo(add) 的返回值就是 wrapper

使用：
    @mygo
def add(a,b):
#不只是可以用来装饰add（），也可以用来装饰add（a，b，c），sayhe（）等所有装饰器，这就是*args和**kwargs的功能
    print(a+b)

add(3,5)
总结：
wrapper
   │
   │ 调用
   ▼
func(*args,**kwargs)
   │
   ▼
add(3,5)
