Python对象可以理解为：
对象 = 一层外壳 + 一个属性字典(__dict__)
所有属性最终都在：__dict__里面。



__getattribute__---总是触发---属性访问总入口 
__getattr__---属性不存在---兜底处理
getattr()---手动调用---安全访问接口

限制属性访问
class Secure:
    def __getattribute__(self, name):
        if name.startswith("_"):
            raise AttributeError("禁止访问私有属性")
        return super().__getattribute__(name)
