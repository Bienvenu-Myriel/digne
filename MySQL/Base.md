
召唤mysql可以通过在cmd或者powershell里输入：mysql -u root -p来实现

基础语法：
1.SQL语句可以单行或者多行书写，以分号（;）结尾。
2.SQL语句可以使用空格/缩进来增强语句的可读性。
3.MYSQL数据库的SQL语句不区分大小写。
4.单行注释用#或--，多行用/* */
5.「MySQL 中，只要空格不破坏关键字、不省略关键字与字符串的分隔，就完全没影响；多余空格 / 换行随便加，不报错。」
