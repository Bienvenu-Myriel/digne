MySQL 条件查询语法（简洁排版版）
1. 比较运算符
= 等于
> 大于
< 小于
>= 大于等于
<= 小于等于
!= / <> 不等于
2. 范围查询
BETWEEN 值1 AND 值2
在两个值之间（包含两端）
NOT BETWEEN 值1 AND 值2
不在两个值之间
3. 指定值匹配
IN (值1, 值2, 值3)
匹配列表中任意一个值
NOT IN (值1, 值2)
不匹配列表中任何值
4. 模糊查询 LIKE
% 匹配任意多个字符
_ 匹配单个字符
LIKE '张%' 以张开头
LIKE '%三' 以三结尾
LIKE '%小%' 包含小
LIKE '_明%' 第二个字是明
5. 空值判断
IS NULL 字段为空
IS NOT NULL 字段不为空
6. 逻辑条件
AND 并且（多条件同时满足）等效于&&
OR 或者（满足一个即可）等效于||
NOT 取反 等效于!
7. 正则匹配
REGEXP '正则表达式'

聚合函数：
COUNT 数个数
SUM 算总和
AVG 算平均
MAX/MIN 找最大最小
GROUP_CONCAT 拼接多行文字
